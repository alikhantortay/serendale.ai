# Serendale AI

## Демо

 https://alikhantortay.github.io/serendale.ai/

## Основной функционал

- Адаптивная вёрстка (Desktop / Tablet / Mobile 320px+)
- Семантическая HTML-разметка
- Современный UI в стиле AI/Web3 лендингов
- Анимации (Framer Motion)
- Базовая интерактивность
- Мобильное бургер-меню
- Секция преимуществ
- Секция отзывов пользователей
- Footer с навигацией и контактами
- JSON-LD микроразметка для отзывов (schema.org)
- SEO-метаданные (Title, Description, OpenGraph)
- Оптимизация изображений (WebP)
- Оптимизация производительности (Lighthouse 90+)
- Деплой на GitHub Pages

## Технологии

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Zustand (state management)
- HTML5 / CSS3

## Производительность

Lighthouse:

- Mobile: 90+ Performance
- Desktop: 100 Performance
- Accessibility: 100
- Best Practices: 100
- SEO: 100

## Оптимизация изображений

- Используется формат WebP
- Адаптивная загрузка hero-изображения
- Preload LCP-ресурса
- Оптимизация размера файлов
